Destructuring

Destructuring

«Destructuring allows properties or values within data structures to be extracted and assigned to variables, simplifying access and improving code readability.»

Destructuring with Data Classes

        
data class User(val id: Int, val name: String, val email: String)

fun main() {
    val user = User(1, "Alice", "alice@example.com")

    // Destructuring declaration
    val (id, name, email) = user

    println("ID: $id")        // Output: ID: 1
    println("Name: $name")    // Output: Name: Alice
    println("Email: $email")  // Output: Email: alice@example.com
}

In this example, destructuring lets us unpack the User class properties into separate variables, enhancing readability and efficiency.

Destructuring in Collections

Example with Lists (similar with Maps)

        
val languages = listOf("Kotlin", "Java", "Python")

val (first, second, third) = languages

println(first)  // Output: Kotlin
println(second) // Output: Java
println(third)  // Output: Python

This demonstrates destructuring on lists, where values are unpacked into individual variables.
