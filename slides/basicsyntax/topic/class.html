Normal class

        
class Test

class Test constructor(variable: Any)

class Test constructor(variable: Any) {
    init{
        val instanceVariable = variable
    }
}

class Person(val name: String, val hasSalary: Boolean) {
    constructor(employee: Employee) : this(employee.name, true)
    constructor() : this("",false)
}

A) This defines a class named Test with no properties or methods. B) This class has a primary constructor that takes a single parameter of type Any. The constructor keyword is optional here. C) This class also has a primary constructor and an init block that initialize a local variable (instanceVariable) with the constructor's parameter. The init block runs every time an instance of the class is created. D) This class has a primary constructor that takes two parameters: name (of type String) and hasSalary (of type Boolean). It has two secondary constructors: One that takes an Employee object and initializes the name with employee.name and sets hasSalary to true. A default constructor that initializes name to an empty string and hasSalary to false.

    is ViewState.Data -> { handleData(viewState.someData)}

Open class

        
open class Base () {
    open fun a() {}
    open val c: Int = 1
    fun b() {}
}

class Derived : Base(){
    final override fun a() {}
    override val c = 3
}

In order to inherit from a class it needs to be open ("open" keyword) To override a function from the parent the function in the parent also needs to be declared "open"

An overridden function is itself is still open. If you want to prohibit this, use final

val basically defines a getter and a setter, so it can be overridden

Abstract class

        
abstract class Base {
    abstract fun a()
    open val c: Int = 1
    fun b() {} // Regular method
}

open class Derived : Base() {
    override fun a() { /* Implementation of abstract method */ }
    override val c = 3 // Overrides property 'c'
}

class SubDerived : Derived() {
    override fun a() {
        super.a()
        println(c)
    }

    override val c: Int
        get() = super.c + 1
}

fun main() {
    val subDerived = SubDerived();
    subDerived.a()
}

To inherit from an abstract class, it must be marked with the "abstract" keyword. Abstract methods and members must be implemented by subclasses. Open properties or methods provide a default implementation that can be overridden.

    }
