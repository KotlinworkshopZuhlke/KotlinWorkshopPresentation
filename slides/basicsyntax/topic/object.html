Companion Object

        
class A {
    companion object {
        fun bar() {
            println("Companion object called")
         }
     }
}

fun main() {
    A.bar()
}

Companion objects are similar to static members in Java. They allow you to define methods and properties that belong to the class rather than to instances of the class.

companion object: Defines a companion object within class A. bar function: A static-like method that can be called without an instance of A.

Anonymus inner class

        
val anonymous = object : Comparator<File> {
    override fun compare(file1: File, file2: File): Int {
        return file1.path.compareTo(file2.path, ignoreCase = true)
    }
}
val files = listOf(File("/Z"), File("/a"))
println(files.sortedWith(anonymous))

An anonymous inner class is a class without a name that is defined at the point of instantiation. It is often used to implement interfaces or extend classes on the fly.

val anonymous: An anonymous inner class implementing Comparator<File>. compare function: Provides a way to compare two File objects by their paths, ignoring case differences. sortedWith: Sorts a list of File objects using the custom comparator defined above.
