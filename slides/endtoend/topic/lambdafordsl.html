Call function with lambda argument

If the last argument of a function is a lambda, when the function is called, the lambda can be moved outside of the parenthesis

        
//function that takes as an argument an int and a lambda
fun isValid(value: Int, validator: (Int) -> Boolean): Boolean {
    return validator(value)
}

val age = 30
val result = isValid(age,  { it in 1..149}) // lambda inside "()"

// last lambda argument can be outside "()"
val result = isValid(age) { it in 1..149}
